Object Challenge

Step 1

Create an array of objects call library. Add 3 objects with a property of title, author, status. Title and author should be strings(whatever value you want) and status should be another object with the properties of own, reading, and read. Which should all be boolean values. For all status, set own to true and reading and read to false.
